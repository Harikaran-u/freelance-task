# Freelancer React App

# overview

Hi, Everyone!!!. Here, it's my freelancer app. Freelancers can see projects and clients can see freelancer details. New users can create profile, signup and old users can login.

**Functionality**

1. Sign-up
2. Login
3. Projects View
4. Freelancer View
5. Profile Update
